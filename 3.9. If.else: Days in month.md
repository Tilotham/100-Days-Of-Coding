## Statement
#### Given a month - an integer from 1 to 12, print the number of days in it in the year 2017.

#### Example input #1
```
1
(January)
```
#### Example output #1
```
31
```
#### Example input #2
```
2
(February)
```
#### Example output #2
```
28
```
## Program
```
a = [31,28,31,30,31,30,31,31,30,31,30,31] 
b = int(input())
b = b-1
print(a[b])
```
